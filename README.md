<div align="center">

  <h1>📦 `MicroservicioStock`</h1>
  <h3> Controla el inventario del restaurante y los ingredientes necesarios para cada plato. </h3>
</div>
* **Gestión de Ingredientes:** Control de cantidades, unidades de medida y niveles de stock.
* **Gestión de ingredientes:** Asociación de ingredientes a platos específicos para definir qué y cuánto se consume al preparar un plato.
* **Consumo y Liberación Automática:** Al ingresar una orden, se descuenta (consume) el stock correspondiente. Si una orden se cancela, el stock se libera y regresa al inventario.
